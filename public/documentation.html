My Movie API Documentation

Overview

The server-side of this API was built to allow users to get information on different movies, their release year, movie genres and directors. Users can also register, creating a user name and password, as well as create a list of their favorite movies.

Errors are returned in the response body with HTTP response codes and messages detailing the error/s.
